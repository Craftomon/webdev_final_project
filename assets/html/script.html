About Me

I started making these little pixel art characters and scenes so I could visualize a scratch project I was working on. I stopped working on it when classes started up again, but I'm not completly done with it, I hope. The development cycle of this project is based entirely on my willingness to continue and how much time I have. Right now, I don't have much of a desire to work on it, but I might come back to it at some point.

About Me

Hello! I’m Craftomon, a pixel artist who creates sprites, tiles, and animated characters for games and digital media. This portfolio showcases my artwork and demonstrates my web development skills.

I started creating pixel art using tools like Aseprite, Piskel, and Procreate. My style is inspired by retro NES, SNES, and Game Boy games.
